# Medical Application and Healthcare Final Project
This repository contains the final project developed for the course **Medical Application and Healthcare**.

## Project Objective
The aim of this project is to analyze the effectiveness of anti-CGPR treatments for migraine patients by comparing objective clinical outcomes 
(Monthly Migraine Days - MMDs), with patient-reported outcomes (HIT-6, MIDAS, HADS).

A specific focus is placed on identifying patients showing a mismatch between biological response and perceived clinical benefit.

## Methods
- Baseline and longitudinal descriptive analyses
- Missing data handling (MICE, LOCF sensitivity analysis)
- Linear mixed-effects models
- Logistic regression for discordant treatment response
